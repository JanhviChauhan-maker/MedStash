# 🏥 MedStash – Healthcare Management System

MedStash is a web-based healthcare management platform built using **Flask** and **SQLite**.  
It provides a complete digital ecosystem for managing patient appointments, doctor consultations, billing, and medical records in a centralized system.

---

## 🚀 Features

### 👩‍💻 Patient Dashboard
- User registration & role-based login
- Book appointments with multiple doctors
- Choose from various medical specialties
- View upcoming appointments
- Access appointment history
- Automatic bill generation after booking
- View and download printable bills
- Track medical records (including weight & height charts)
- View billing history

### 👨‍⚕️ Doctor Dashboard
- View today’s appointments
- View upcoming appointments
- Access appointment history
- Manage consultations
- Issue digital prescriptions
- Access patient medical records
- Billing management
- Wallet overview

---

## 🛠 Technology Stack

- **Backend:** Flask (Python)
- **Database:** SQLite
- **Frontend:** HTML, CSS, Bootstrap
- **Authentication:** Role-based login system

---

## 🎯 Project Objective

The goal of MedStash is to digitize hospital workflows by integrating appointment booking, billing, prescriptions, and patient record management into one structured platform.
